Backups

Backing up and restoring data is done easily with the mongodump and mongorestore tool. These tools should not be confused with mongoexport and mongoimport, which allow you to load a import and export data in JSON format. (Not ideal for backups, due to loss of information - JSON has fewer types)

The mongodump command could look like this:


mongodump \
--host 188.166.100.89 \
--port 27017 \
-u administrator \
-p pass \
--gzip \
--archive=backup.$(date +%Y_%m_%d_%H%M%S).gz

You could set this up as a daily job in Jenkins for a very simple backup system.
